
# Yazi Quest - Level Design Document

This document outlines the 17 levels of Yazi Quest, their objectives, and the concepts they teach.

## Episode 1: Awakening (Basics)

**Level 1: System Navigation**
- **Concept:** Basic Movement (`j`, `k`, `h`, `l`)
- **Task:** Navigate to specific directories.

**Level 2: Threat Elimination**
- **Concept:** Deletion (`d`)
- **Task:** Identify and delete a "virus" file.

**Level 3: Asset Relocation**
- **Concept:** Cut & Paste (`x`, `p`)
- **Task:** Move a file from one directory to another.

**Level 4: Protocol Design**
- **Concept:** Creation (`a`)
- **Task:** Create directories and files.

**Level 5: Batch Deployment**
- **Concept:** Batch Selection (`Space`)
- **Task:** Select multiple files and move them simultaneously.

## Episode 2: Fortification (Advanced & Time Pressure)

**Level 6: Intelligence Gathering**
- **Concept:** Filtering (`f`)
- **Task:** Use filter to find a hidden key file.

**Level 7: Deep Scan Protocol**
- **Concept:** Fuzzy Find (`Z`)
- **Task:** Jump to distant directories without manual navigation.

**Level 8: Neural Construction & Vault**
- **Concept:** Complexity Management
- **Task:** Create a complex nested structure and copy files within a time limit.

**Level 9: Stealth Cleanup**
- **Concept:** Visual Selection
- **Task:** Select multiple temp files and delete them in one go.

**Level 10: Encrypted Payload**
- **Concept:** Archives
- **Task:** Enter a `.zip` file (treated as a directory in Yazi) and extract content.

**Level 11: Live Migration**
- **Concept:** Advanced Movement
- **Task:** Move files out, modify structure (implied), and move them back.

## Episode 3: Mastery (Efficiency & Keystroke Limits)

**Level 12: Identity Forge**
- **Concept:** Renaming (`r`)
- **Task:** Rename files to mimic system processes.

**Level 13: Root Access**
- **Concept:** Precision
- **Task:** Manipulate system files with strict keystroke limits.

**Level 14: Shadow Copy**
- **Concept:** Directory Copying
- **Task:** Copy entire directory trees.

**Level 15: Trace Removal**
- **Concept:** Cleanup
- **Task:** Delete logs and return to root efficiently.

**Level 16: Grid Expansion**
- **Concept:** Deep Nesting
- **Task:** Build deep directory structures quickly.

**Level 17: System Reset**
- **Concept:** Final Exam
- **Task:** Wipe everything except one specific directory.
